# Normas Urbanísticas — Fe de Erratas, Modificación del Plan Regulador Comunal de Rancagua "Tramitación N° 16", Zona EX 6

**Fuente:** Aviso S/N, Fe de Erratas, Municipalidad de Rancagua, publicación en el Diario Oficial 05-jul-2010 (corrige publicación del 26-feb-2009, página 8)
**Extracción:** páginas 1 y 2 de 2 del documento. Texto nativo completo. No incluye cuadro numérico de COS/coeficiente de constructibilidad/altura — solo usos de suelo.

## EX 6 (Zona de Extensión Urbana 6)

Fe de erratas al numeral 2.5.4: se corrige el listado de usos de suelo permitidos y prohibidos de la Zona EX 6.

### Usos de suelo (versión corregida — "DEBE DECIR")

| Tipo de uso | Permitidos - Prohibidos |
|---|---|
| Residencial | Permitido, con excepción de lo indicado como prohibido |
| Comercio | Permitido, salvo lo indicado como prohibido |
| Culto y Cultura | Permitido |
| Deporte | Permitido, salvo lo indicado como prohibido |
| Educación | Permitido, solo Parvularia y Básica (versión anterior "DICE" no tenía esta restricción — la corrección la introduce) |
| Salud | Permitido, con excepción de lo indicado como prohibido |
| Seguridad | Permitido, con excepción de lo indicado como prohibido |
| Servicios | Permitido, con excepción de lo indicado como prohibido |
| Social — Sedes de Juntas de Vecinos | Permitido |
| Espacio Público | Permitido |
| Área Verde | Permitido |
| Residencial — Hospederías, Residenciales, Moteles | Prohibido |
| Científico | Prohibido |
| Comercio — Mercados, Ferias Libres, Centros de Servicio Automotor o Servicentros, Locales de Expendio de Combustibles, Playas de Estacionamiento, Cabaret, Quintas de Recreo, Bares, Discotecas, Depósito de Alcoholes | Prohibido |
| Deportes — Estadios, Centro y Clubes Deportivos | Prohibido |
| Esparcimiento | Prohibido |
| Salud — Hospitales, Cementerios y Crematorios | Prohibido |
| Seguridad — Bases Militares, Cárceles, Centros de Detención y Otros | Prohibido |
| Servicios — Talleres Mecánicos de Reparación Automotriz, Vulcanizaciones, Garajes, Carpintería Metálica, Taller de Vulcanización, Imprentas y Litografías, Tapicerías, Ebanistería, Tornería, Cerrajería Artística, Amasanderías (salvo lo permitido) | Prohibido |
| Actividades Productivas | Prohibido |
| Infraestructura de Transporte | Prohibido |
| Infraestructura Sanitaria | Prohibido |
| Infraestructura Energética | Prohibido |

**Nota:** el único cambio sustantivo identificado entre la versión "DICE" (26-feb-2009) y "DEBE DECIR" (05-jul-2010) es la restricción del uso Educación de "sin límite" a "solo Parvularia y Básica". El resto de los usos permitidos/prohibidos es idéntico en ambas versiones.

### Condiciones de subdivisión y edificación

No especificado — este documento (fe de erratas) no incluye cuadro numérico de COS, coeficiente de constructibilidad, altura, predial mínima, agrupamiento, antejardín ni densidad para la Zona EX 6; solo corrige el listado de usos de suelo.
