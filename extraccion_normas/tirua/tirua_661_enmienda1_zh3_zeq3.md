## Nota de alcance

La enmienda sustituye los cuadros de usos de suelo y normas urbanísticas de las zonas ZH3 y ZEQ3, ambas con "sector con riesgo de tsunami" (aplica artículo 9 de la Ordenanza).

---

## ZH3 (Zona Habitacional)

### Usos de suelo

| Tipo de uso | Permitido - Prohibido |
|---|---|
| Residencial | Permitido |
| Equipamiento: Científico | Prohibido |
| Equipamiento: Comercio | Permitido, excepto grandes tiendas |
| Equipamiento: Culto y cultura | Permitido |
| Equipamiento: Deporte | Permitido, excepto estadios y autódromos |
| Equipamiento: Educación | Prohibido |
| Equipamiento: Esparcimiento | Permitido, excepto parque de entretenciones y parque zoológico |
| Equipamiento: Salud | Prohibido |
| Equipamiento: Seguridad | Prohibido |
| Equipamiento: Servicios | Prohibido |
| Equipamiento: Social | Permitido |
| Actividades Productivas | Prohibido, excepto inofensivas |
| Infraestructura: Transporte | Permitido, solo terminal de transporte terrestre |
| Infraestructura: Sanitaria | Prohibido |
| Infraestructura: Energética | Prohibido |
| Espacio Público | Permitido |
| Área Verde | Permitido |

### Normas urbanísticas

| Norma | Valor |
|---|---|
| Superficie de subdivisión predial mínima | 300 m² |
| Coeficiente de ocupación de suelo | 0,6 |
| Coeficiente de constructibilidad | 2,0 |
| Altura máxima de edificación | 9,0 m |
| Sistemas de agrupamiento | Aislado y pareado |
| Densidad máxima | 70 hab/há |

Zona con sector con riesgo de tsunami. Aplica artículo 9 de esta ordenanza.
