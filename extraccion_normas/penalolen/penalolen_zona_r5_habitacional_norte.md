## Zona R5 (Zona Habitacional Norte)

### Usos de suelo

| Uso | Detalle |
|---|---|
| Permitido | a) Vivienda |
| Permitido — Equipamiento (condicionado) | Se permitirá equipamiento de nivel Vecinal solo en los predios que enfrentan Av. Antipurén, hasta una profundidad de 50 m, y solo los tipos señalados a continuación |
| Permitido — Salud | Centros médicos y dentales, policlínicos, hogares de ancianos, casa de salud y reposo |
| Permitido — Educación | Educación básica, educación pre-básica, jardines infantiles, sala cuna |
| Permitido — Seguridad | Retenes |
| Permitido — Culto | Capillas |
| Permitido — Cultura | Bibliotecas, casa de la cultura, auditorios, teatros, salas de concierto |
| Permitido — Organizaciones Comunitarias | Sedes juntas de vecinos, sedes centros de madres, sedes sociales |
| Permitido — Áreas Verdes | Plazas, jardines infantiles, jardines |
| Permitido — Esparcimiento y Turismo | Cines, fuentes de soda, bares |
| Permitido — Comercio | Locales comerciales |
| Permitido — Servicios Públicos | Correos |
| Permitido — Servicios Artesanales | Talleres artesanales, talleres de reparación, lavandería, carpintería, mueblería, peluquería, gasfitería, jardines de plantas |
| Excluido | Todos los no indicados expresamente como permitidos |

### Normas Específicas — Vivienda

| Parámetro | Valor |
|---|---|
| Superficie predial mínima | 800 m² |
| Frente predial mínimo | 20 m |
| Sistema de agrupamiento | Aislado |
| Adosamiento | No se permite |
| Coeficiente de ocupación de suelo | 0,2 |
| Coeficiente de constructibilidad | 0,5 |
| Altura máxima de edificación | 9 m |
| Densidad bruta máxima | 50 hab/há |
| Rasante | Según OGUC |
| Distanciamiento | 3 m (con o sin vano) |
| Antejardín | 5 m |
