Fuente: Ficha municipal de zona, 1 página escaneada, verificada visualmente.

## Zona CC

### Usos de suelo

| Uso | Detalle |
|---|---|
| Permitido | Equipamiento, con excepción de los indicados en usos excluidos |
| Excluido | Vivienda, salvo la del cuidador |
| Excluido — Equipamiento Salud | Escala Comunal: cementerios, plantas y botaderos de basura |
| Excluido — Equipamiento Cultura | Escala Regional-Interurbana: zoológicos |
| Excluido — Equipamiento Esparcimiento y Turismo | Escala Regional-Interurbana: hipódromos |
| Excluido — Equipamiento Comercio Minorista | Escala Regional-Interurbana: centros comerciales, supermercados, grandes tiendas, terminales de distribución; Escala Comunal: centros comerciales, mercados, supermercados, ferias libres, locales comerciales, servicentros, playas y edificios de estacionamiento; Escala Vecinal: locales comerciales, ferias libres, playas de estacionamiento |
| Excluido — Servicios Artesanales | Escala Comunal: garaje, talleres; Escala Vecinal: talleres molestos |
| Excluido — Transporte | Escala Regional-Interurbana: terminales agropecuarios, terminales pesqueros |
| Excluido — Industria y Almacenamiento | Escala Regional-Interurbana: inofensivos molestos, insalubres |
