## ZRE (Zona Especial Residencia y Equipamiento)

### Usos de suelo

| Tipo de uso | Permitidos - Prohibidos |
|---|---|
| Residencial | Permitido |
| Equipamiento — Culto y Cultura | Permitido: solo aquellos destinados al desarrollo espiritual-religioso; centros culturales y bibliotecas |
| Equipamiento — Comercio | Permitido: solo centros y locales comerciales |
| Equipamiento — Servicios | Permitido |
| Espacio Público | Permitido |
| Área Verde | Permitido |
| Todo lo no indicado expresamente como permitido | Prohibido |

### Condiciones de subdivisión y edificación

| Norma | Valor |
|---|---|
| Superficie de Subdivisión Predial Mínima | 200 m² |
| Densidad bruta máxima | 280 hab/há |
| Coeficiente de Ocupación de Suelo | 0,6 |
| Coeficiente de Constructibilidad | 2 |
| Sistemas de Agrupamiento — Uso Residencial | Aislado, sin adosamiento en predios de superficie superior a 500 m² |
| Sistemas de Agrupamiento — Otros Usos | Aislado, sin adosamiento |
| Distanciamientos — Uso Residencial | Sobre 9 metros de altura de edificación: 6 metros mínimo |
| Distanciamientos — Uso Equipamiento | Sobre 3,5 metros de altura de edificación: 4 metros mínimo |
| Altura Máxima de Edificación — Uso Residencial | 12 metros |
| Altura Máxima de Edificación — Otros Usos | 9 metros |
| Antejardín | 3 metros mínimo |
