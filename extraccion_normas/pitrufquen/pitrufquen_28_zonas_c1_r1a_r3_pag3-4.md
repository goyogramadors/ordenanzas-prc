# Normas Urbanísticas — Decreto 28, Modificación del Plan Regulador Comunal de Pitrufquén, Zonas C-1, R1-A, R3

**Fuente:** Decreto 28, Ministerio de Vivienda y Urbanismo, 2012 — Ley Chile / BCN
**Extracción:** páginas 3 y 4 de 6 del documento. **Trampa mixta confirmada:** el texto solo dice "...serán las siguientes:" y la tabla real está incrustada como imagen JPEG (verificado con pdfimages y por render). ## C-1 (Zona Habitacional Mixta)

### Usos de suelo

| Tipo de uso | Permitidos - Prohibidos |
|---|---|
| Residencial | Permitido |
| Equipamiento — Científico, Comercio, Culto y Cultura, Deporte, Educación, Esparcimiento, Servicios, Social | Permitido |
| Equipamiento — Salud | Permitido, excepto Cementerios y crematorios |
| Equipamiento — Seguridad | Permitido, excepto Cárceles, centros de detención, centros de internación provisoria, centros cerrados de privación de libertad |
| Actividades Productivas — Industria inofensiva | Permitido |
| Actividades Productivas — Bodegaje y talleres inofensivos | Permitido |
| Todos los usos no señalados como permitidos | Prohibido |

### Condiciones de subdivisión y edificación

| Norma | Valor |
|---|---|
| Superficie de subdivisión predial mínima | 250 m² |
| Superficie predial mínima — Equipamiento y Talleres | 250 m² |
| Superficie predial mínima — Industria y Bodegaje | 500 m² |
| Coeficiente de ocupación de suelo — Residencial, Equipamiento y Talleres | 0,4 |
| Coeficiente de ocupación de suelo — Industria y Bodegaje | 0,6 |
| Coeficiente de constructibilidad | 0,8 |
| Altura máxima de edificación | 7 metros |
| Sistema de agrupamiento | Aislado |
| Adosamiento | Se prohíbe para actividades productivas |
| Distanciamiento — Residencial, Equipamiento y Talleres | Según OGUC |
| Distanciamiento — Industria y Bodegaje | 3 metros |
| Antejardín | 5 metros |
| Densidad bruta máxima | 200 hab/ha |

## R1-A (Área R1-A: Zona inundable o potencialmente inundable, agregada a continuación del Área R2)

### Usos de suelo

| Tipo de uso | Permitidos - Prohibidos |
|---|---|
| Área Verde | Permitido |
| Todos los usos no señalados como permitidos | Prohibido |

### Condiciones de subdivisión y edificación

| Norma | Valor |
|---|---|
| Superficie de subdivisión predial mínima | 2.500 m² |
| Coeficiente de ocupación de suelo | 0,2 |
| Coeficiente de constructibilidad | 0,2 |
| Altura máxima de edificación | 7 metros |
| Sistema de agrupamiento | Aislado |
| Adosamiento | Se prohíbe |
| Distanciamiento | 5 metros |
| Antejardín | 10 metros |
