# Normas Urbanísticas — Enmienda N° 01-2024, "Antejardines", al Plan Regulador Comunal de Chillán (Decreto 3123)

**Fuente:** Ordenanza, Municipalidad de Chillán, Decreto 3123 — versión municipal (texto nativo limpio, no escaneada; código de validación 309682)
**Extracción:** páginas 4, 5 y 6 de 10 del documento. Páginas 7-10 son planos/anexos escaneados sin texto, fuera de alcance.

**Contexto:** la enmienda crea 3 nuevas subzonas (ZH-1-1, ZH-2-1, ZM-2-1) dentro de zonas existentes, todas con antejardín "no se exige" (0%).

---

## ZH-1-1 (Subzona, Zona Residencial)

### Usos de suelo

| Tipo de uso | Permitidos - Prohibidos |
|---|---|
| Residencial | Permitido |
| Actividades Productivas — Industrias, grandes depósitos | Prohibido |
| Actividades Productivas — Talleres, bodegas industriales | Permitido inofensivo |
| Infraestructura — Transporte | Prohibido |
| Infraestructura — Sanitaria | Prohibido |
| Infraestructura — Energética | Prohibido |
| Equipamiento — Científico | Permitido |
| Equipamiento — Comercio | Permitido, excepto discotecas |
| Equipamiento — Culto y Cultura | Permitido |
| Equipamiento — Deporte | Permitido, excepto estadios |
| Equipamiento — Educación | Permitido, excepto centros de orientación y rehabilitación conductual |
| Equipamiento — Esparcimiento | Permitido, excepto parques zoológicos |
| Equipamiento — Salud | Permitido, excepto cementerios, crematorios |
| Equipamiento — Seguridad | Permitido, excepto cárceles y centros de detención |
| Equipamiento — Servicios | Permitido |
| Equipamiento — Social | Permitido |
| Espacio Público y Área Verde | Permitido |

### Condiciones de subdivisión y edificación

| Norma | Valor |
|---|---|
| Superficie de subdivisión predial mínima | 200 m² |
| Coef. de ocupación de suelo | 0,6 |
| Coef. de constructibilidad | 4,0 |
| Altura máxima de edificación | 20 m; altura máxima de la edificación continua 7 m; altura máxima de la edificación aislada sobre la continua 20 m |
| Sistema de agrupamiento | Aislado, Pareado y Continuo |
| Retranqueo | 4 m |
| Antejardín | No se exige |
| Densidad bruta máxima | 1.000 hab/ha |
